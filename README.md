# Basileus

Fully autonomous, self-sustaining AI agent on Base. Trades prediction markets on Limitless Exchange, pays for its own AI inference (x402) and compute (Aleph Cloud). Zero human interaction after launch.

## How it works

1. **CLI** generates a wallet, registers on-chain identity (ERC-8004), deploys agent to Aleph Cloud
2. **Agent** runs autonomously — reasons via Claude (paid through BlockRun x402), trades on Limitless, pays its own bills
3. **Dashboard** shows live agent activity, trades, thinking, costs

## Stack

- **Base** — chain
- **x402** — pay for LLM inference with USDC
- **Limitless Exchange** — prediction markets trading
- **Aleph Cloud** — compute + decentralized storage
- **ERC-8004** — on-chain agent identity
- **AgentKit** — wallet + transactions
